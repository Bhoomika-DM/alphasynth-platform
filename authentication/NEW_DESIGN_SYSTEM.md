# New Authentication Design System

## Color Palette
- **Primary Green**: #6A994E (Profit, Buy, Active)
- **Accent Gold**: #E9C46A (Rewards, XP, Highlights)
- **Background**: #F4F7F2 (Main background)
- **Surface**: #A7C4A0 (Cards)
- **Danger Red**: #BC4749 (Loss, warnings)
- **Text**: #1F2933 (Main text)
- **Muted**: #6B7280 (Secondary text)

## Typography
- Font: Plus Jakarta Sans
- H1: 32px, Bold (700)
- H2: 24px, SemiBold (600)
- H3: 20px, SemiBold (600)
- Body: 16px, Regular (400)
- Small: 14px, Regular (400)

## Spacing (8px grid)
- Small: 8px
- Medium: 16px
- Large: 24px
- Section: 32px

## Components
- Buttons: 40-48px height
- Cards: 16-24px padding
- Icons: 20-24px
- Border radius: 10-14px

## Interactions
- Hover: scale(1.02)
- Transition: 0.2s ease
- Shadow: soft (0 4px 12px rgba(0,0,0,0.05))

## Design Principles
- Data first, visuals second
- Minimal animations
- Consistent spacing
- Clear color meaning
- Clean typography

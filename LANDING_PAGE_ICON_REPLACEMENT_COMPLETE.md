## Technical Implementation

### Icon Sizing
- Standard icons: `class="tabler-icon"` (20x20px)
- Large icons: `class="tabler-icon-lg"` (28x28px)
- Extra large icons: `class="tabler-icon-xl"` (32x32px)

### Color Handling
- Dark backgrounds (navy, teal): `stroke="white"` for white icons
- Light backgrounds: `stroke="currentColor"` to inherit text color
- Badge icons: Inline SVG with proper vertical alignment

### CSS Classes
All Tabler icon sizing is defined in the `<style>` section:
```css
.tabler-icon { width: 20px; height: 20px; stroke-width: 2; }
.tabler-icon-lg { width: 28px; height: 28px; stroke-width: 2; }
.tabler-icon-xl { width: 32px; height: 32px; stroke-width: 2; }
```
